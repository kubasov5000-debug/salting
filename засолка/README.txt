
    # ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ ПРИЛОЖЕНИЯ "ДОМАШНЯЯ ЗАСОЛКА 2.0"
    
    ## 1. ЗАПУСК ПРОГРАММЫ
    - Установите Python 3.6 или выше
    - Установите библиотеку Pillow: pip install Pillow
    - Запустите программу: python pickling_app.py
    
    ## 2. ДОБАВЛЕНИЕ ИЗОБРАЖЕНИЙ
    - Создайте папку "images" рядом со скриптом
    - Добавьте изображения с именами, соответствующими полю "image" в рецептах
    - Или нажмите "Выбрать папку" для выбора другой папки
    
    Пример имен файлов:
    - cucumbers.jpg
    - tomatoes.jpg
    - cabbage.jpg
    - salo.jpg
    - mushrooms.jpg
    и т.д.
    
    ## 3. ИСПОЛЬЗОВАНИЕ
    - Выберите рецепт из списка слева
    - Просмотрите ингредиенты и способ приготовления
    - Сохраните понравившийся рецепт в файл
